## Prep Work
### Get into your "Work" folder

*Watch in the file explorer if you want, in your WORK folder*

F12 / Ctrl+Alt+T (Open terminal)

`cd `

`cd WORK` (whatever you called it when you installed Vrk)

`mkdir shell`

`mkdir xml`

`mkdir html`

`ls`

Each of these folders is for different projects. Open one for each.

##### SH script

`gedit head.sh`

Copy and paste this in the top.

*Ctrl+C, Ctrl+V*

`#!/bin/sh`

Close

*Alt+F4*

`ls`

##### XML

`gedit head.xml`

Copy and paste this in the top.

*Ctrl+C, Ctrl+V*

`<?xml version="1.0" encoding="UTF-8"?>`

Close

*Alt+F4*

`ls`

##### HTML

`gedit head.html`

Copy and paste this in the top.

*Ctrl+C, Ctrl+V*

`<!DOCTYPE html>`

`<head>`

`<title></title>`

`</head>`

`<body>`

` `

`</body>`

Close

*Alt+F4*

`ls`

### Move files

Move everything into place

`mv head.sh shell`

`mv head.xml xml`

`mv head.html html`

`ls`
